About Me & Timeline

I am a passionate Gameplay Programmer and software enthusiast based in Ottawa. My focus is on writing clean, efficient code that powers fun player mechanics and responsive game feel. Whether it's procedural world generation in 2D grids, real-time AI state machines, or custom UI systems, I love solving the technical puzzles that make games tick.

I actively participate in hackathons and game jams, collaborating with developers and artists to build complete game releases under tight constraints. I specialize in Godot Engine (GDScript) and Pygame (Python), alongside core web technologies.
